EBA Command Center 9.2-beta1 Released!

We've released EBA Command Center 9.2-beta1 to our Beta download section!

We've fixed numerous bugs that we mentions previously, such as:

  • Crazy lag when downloading files.
  • High CPU usage when setting languages.
  • "err.undefined" errors.
  • Debug Command
  • Crazy lag when creating a restore point.

This update includes the following features:

  • Language Support
  • Commands: Debug, Language, Restore
  • Password Encryption
  • Progress Bar on longer actions
  • Automatic creation of System Restore Points
  • Everything else listed in the Changelog

EBA Command Center 9.2

We're going to delay EBA Command Center 9.2 betas. We're going to fix up a few things, then we'll release a beta. Here's what we'll be doing:

  • We'll be fixing the "err.undefined" errors that we know of.
  • Adding account config and default shutdown method.
  • Importing and Exporting.

All features listed in the changelog file will be available (except the debug command).

EBA Command Center 9.2-beta1 should be available within 3 days.

EBA Command Center 9.2

We're going to release the first public beta of EBA Command Center Release 1.2. Most features will be available in Release 1.2-beta1, but the following features will be unavailable:

  • Some prompts might result in the text "err.undefined", which simply means there is no text for that box yet. There should be very few of these.
  • Changing account settings or the default shutdown method.
  • Most config options will result in an err.undefined.
  • Exporting will not work.
  • Importing will only work by drag-and-drop. The command currently does not function.
  • Changing the Modifiable Startup Settings using the Debug command.
