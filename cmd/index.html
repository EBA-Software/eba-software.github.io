Frequently Asked Questions

What is EBA Command Center?

EBA Command Center is a Windows shortcut application. It is designed to clean the clutter on your desktop, by allowing you to move shortcuts off your desktop, and link to them using 'commands'. EBA Command Center can link a 'command' to a file, folder, website, or command prompt.

Which edition of EBA Command Center do I need?
