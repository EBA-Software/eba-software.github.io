Announcement | EBA Command Center 10

EBA Command Center 10 is the new EBA Command Center! EBA Command Center 10 takes everything to a whole new level!

EBA Command Center is now an EXE!

EBA Command Center itself is no longer a .vbs file. It is now an actual EXE file, with a custom user-interface, and many more possibilities than we can even count!

Custom UI

EBA Command Center no longer follows a boring UI. EBA Command Center's UI has been remade fully, with support for various themes, and a quick access toolbar.

EBA Software Accounts!

EBA Command Center 10 comes with a new feature, EBA Software Accounts! These accounts (in the future) can be used across multiple EBA Software programs, and can be used to backup and sync your data.

Themes

EBA Command Center 10 adds themes to the new UI. You can pick from:

  1. Light Theme (default) - A light footer and background with a black logo.
  2. Dark Theme - A dark footer and background with a grey logo.
  3. EBA Theme - A dark-blue footer, white background, with a black logo. Made to match the EBA Software website.
  4. Dark EBA Theme - A blue footer, dark-blue background, with a grey logo. Made to look similar to the EBA Software website.
  5. Custom - Custom colors! Customize the background, footer, and logo colors.

EBA Command Center for Educators!

EBA Command Center has a new edition for educators! This edition is not free, but it offers educator-exclusive features, such as:

Custom Blocklist

Educators can create a custom blocklist. This prevents students from creating commands that match the blocklist.

Force Settings

Educators can prevent students from changing malware detection, creating new commands, logging, and anything else found in the config menu. Educators can lock any setting.

Lock Students Out

Educators can force a student to sign in to their User Account to use commands, and educators can block a student temporarily.

Special Admin Account

Educators can have special account types. This allows a district administrator to give some (but not full) access to some settings.

More to come soon!

EBA Command Center 10 has more surprises coming soon!

When will EBA Command Center 10 be available!

Right now, we don't know for sure. At the moment, we're working on backwards compatibility, Windows ME/2000 support, and other various features. We are hoping to begin betas in mid 2022, and release in mid-late 2022. Please be patient while we work on improving EBA Command Center 10!
