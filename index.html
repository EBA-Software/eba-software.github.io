EBA Software

Announcement

On April 29, 2022, we made the decision to completely stop development for EBA Command Center. While the project has became a much bigger thing than it's original intension was, we have made the decision to stop development. EBA Command Center 10 is not in a stable condition and will not be released unless we end up resuming development in the future. Thank you for supporting EBA Software and EBA Command Center. We hope to see you soon.

EBA on Twitch!

EBA Software is ending support for all applications soon! If you're interested in the developers of EBA Software, great news! I (the lead developer) stream on Twitch! Highly recommend checking it out.

Frequently Asked Questions

Is EBA Software done developing software?

At the moment, yes. All development has been stopped and we do not intend to return. However, there is a chance of returning in the future.

How can I get updated about all things related to EBA Software?

Updates related to EBA Software can be found on the EthanBAlarms Discord Community!

What is EBA Software?

EBA Software is a small software development team. We create programs for Windows that make it easier to use your PC.

What is EBA Software Gaming?

EBA Software Gaming is a new branch of EBA Software. This branch releases games for Windows and Android. You can check out everything made by EBA Software Gaming using the EBA Software Gaming link above.

How many programs have you made?

We've made several small programs, but only two are available to the public. We're working on adding our other programs into EBA Command Center 10!
