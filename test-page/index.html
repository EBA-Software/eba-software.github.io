Announcement

On December 21, 2021, we made the decision to completely stop development for EBA Application Manager, EBA Database, and EBA Shortcut Manager. All of these programs and their functions will be merged with EBA Command Center 10.

EBA Command Center 10:

We're working on the biggest update to EBA Command Center we've ever had! EBA Command Center 10 will include major changes.

We're not releasing any of our plans quite yet, as it is very early in development, and some of these features might not make the cut. Once we're ready, we'll release our plans.

For now, expect "The New EBA Command Center" to release somewhere between March 2022 and June 2022. Expected release date may change.

Frequently Asked Questions

What is EBA Software?

EBA Software is a small software development team. We create programs for Windows that make it easier to use your PC.

What is EBA Software Gaming?

EBA Software Gaming is a new branch of EBA Software. This branch releases games for Windows and Android. You can check out everything made by EBA Software Gaming using the EBA Software Gaming link above.

How many programs have you made?

We've made several small programs, but only two are available to the public. We're working on adding our other programs into EBA Command Center 10!
