Installing EBA Command Center

EBA Command Center must be installed before you can use it. Lets take a look at this process.

End-User License Agreement

In many versions of EBA Command Center, you'll be required to click Yes to agree to the EULA (or in older versions, Terms of Service).

This is always the first step in versions that include it.

Confirmation 1

This is the first confirmation. If EBA Command Center was not found on your device, this confirmation simply alerts you of that.

Installation Directory

You'll be asked where you want to install EBA Command Center. The default directory is always C:\Program Files\EBA\Cmd. If an installation of EBA Command Center is found at the selected directory, it'll switch to update mode.

Portable Mode

This is where you choose whether or not you want to use Portable Mode.

Confirmation 2

This is the second confirmation. This gives you all of the details for your installation (including directory, portable mode...). If you selected portable mode, this is the final confirmation.

User Account Control

If you selected non-portable mode, you'll get a UAC prompt to proceed with installation. This is the final confirmation.
