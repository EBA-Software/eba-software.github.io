The Config Menu

EBA Command Center includes a configuration menu. To access this menu, you must be logged in as a level 2 user. This menu is access by running the "config" command.

Dev Info: You can also use any command that links to execution value "eba.config".

Command Tree:

Command Syntax 1: config {1} {2} {3}

Command Syntax 2: config {1} {2}

Command Syntax 3: config {1}

Command Syntax 4: config

  • Config - The command used to access the menu.
    • Cmd - The command used to access the Command Editor. ({1})
      • {command-name} - Command to edit/create. ({2})
      • If command exists:
        • Target - Edit the command target. ({3})
          • {new-value} - New target.
        • Admin - Change "Run as Administrator"
          • {new-value} - New value.
        • Login - Change perms required.
          • {new-value} - New level requirement.
        • Delete - Delete the command.
      • Otherwise:
        • {type} - Command type. ({3})
          • {target} - Command target.
        • Exe - Command type EXE (hidden type). ({3})
          • {target} - Command target.
        • Import - Import from a Windows Shortcut File. ({3})
          • {target} - Command target.
    • Acc - The command used to access the Account Editor. ({1})
      • {user-name} - Username to edit/create. ({2})
      • If user exists:
        • Pword - Edit the account password. ({3})
          • {old-value} - Required. Old password.
          • {new-value} - Required. New password.
        • Perms - Change user perms.
          • {new-value} - New level.
        • Delete - Delete the user.
      • Otherwise:
        • {password} - Required. Create password. ({3})
        • {perms} - Required. Set level.
    • Logs - The command used to toggle logging. ({1})
      • Prompts a Yes/No box. {2} can be True or False.
    • Save - The command used to toggle saveLogin. ({1})
      • Prompts a Yes/No box. {2} can be True or False.
    • Shut - The command used to change defaultShutdownMethod. ({1})
      • {value} - New shutdown method. ({2})
    • Time - The command used to change shutdownTimer. ({1})
      • {value} - New shutdown timer. ({2})
    • Adv - The command used to open RegEdit and view advanced settings. ({1})
